# 📱 WhatsApp Weather Automation App

An Android application that automates sending WhatsApp messages using voice commands.

## 🚀 Features
- 🌦️ Real-time weather updates
- 📍 Sends live location
- 🕒 Includes date & time
- 🎙️ Voice assistant integration

## 🛠️ Tech Stack
- Kotlin
- MVVM Architecture
- Weather API
- Location Services
